# Benchmark reports

This folder stores reviewable offline benchmark outputs.

Phase 3 introduces corpus coverage summaries here so the accepted replay count per cohort and matchup is always visible.

Phase 4 adds resumable extraction run summaries and failure logs:

- `batch-run-summary.json`
- `batch-run-summary.md`
- `failed-replays.jsonl`

Phase 5 adds benchmark-analysis reports:

- `benchmark-summary.json`
- `benchmark-summary.md`

Internet sourcing research and discovery outputs may also be stored here, for example:

- SpawningTool discovery JSON snapshots
- internet download summary JSON outputs
