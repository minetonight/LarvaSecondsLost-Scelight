Larva Seconds Lost

This external module now includes the Epic 2 fallback replay-view surface.

What is proven in this build:

  • The module compiles against the Scelight external module API.
  • The module can package a manifest, jar, and deployment zip.
  • The module can contribute visible UI inside Scelight once enabled.
  • Lifecycle logging is available for startup and shutdown verification.
  • The module can analyze a replay on its own page, either by manual file selection or from the Replay Folder Monitor event stream.
  • The module now renders a first chart-like timeline preview using a replay-derived placeholder interval.
  • The module now reports that native chart dropdown registration is not exposed by the public external module API.
  • The module now reports that native Base Control chart augmentation is also not exposed by the public external module API.

Direct injection into Scelight's internal replay analyzer page is not exposed by the public external module API, so replay diagnostics currently live on the module-owned Larva page.

Real larva windows are still deferred to later epics, but the page layout is now ready to host them.

The native chart dropdown is currently treated as unsupported for a pure external module because Scelight builds it from internal chart enums and factories that are not part of the external API.

The native Base Control chart is also currently treated as unsupported for augmentation because it is created from those same internal chart classes and no public augmentation callback is exposed.
